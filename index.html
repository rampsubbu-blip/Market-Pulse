Install Notes

This PWA is designed to behave properly on iPhone when hosted over `https://`.

On iPhone: open the hosted URL in Safari, tap Share, then tap Add to Home Screen.

For Chrome and Edge on desktop or Android, an install prompt may appear automatically.

When a source blocks live browser access, the app keeps your most recent successful snapshot available offline.

Fetch Notes

Some sources are protective about cross-origin requests. The dashboard retries using browser-safe proxy fallbacks before giving up.
